## 测试方法

### 1. 页面导航测试

使用 `agent-browser` 工具进行自动化导航测试：

```bash
# 测试首页导航
agent-browser open http://localhost:3000
agent-browser snapshot -i
agent-browser click @e1  # 点击导航链接
agent-browser wait --load networkidle
agent-browser get url    # 验证URL是否正确

# 测试表单提交
agent-browser fill @e2 "test@example.com"
agent-browser click @e3
agent-browser wait --url "**/success"
```

### 2. 内容准确性检查

- 验证页面内容是否符合逻辑
- 检查链接是否指向正确的页面
- 确认所有动态内容正确加载
- 验证法律信息的准确性

### 3. UI设计指南合规性

使用 `web-design-guidelines` 技能检查：

- 响应式设计
- 可访问性
- 视觉一致性
- 交互体验
- 性能优化

### 4. 功能测试

- 测试核心功能的完整流程
- 验证错误处理
- 测试边界情况
- 检查数据验证
